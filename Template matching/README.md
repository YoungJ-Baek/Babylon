# Template matching
In this folder, I introduce template matching based on image labeling

You can change threshold value while considering trade-off between accuracy and complexity. In this code, you can see image labeling function. You should get the function from image labeling folder. Also, the function named 'Find_Region' is simple function to make label image space. You can make one, or you can use codes in image labeling folder.
